Php Unit

Utile :

PhpUnit
Cet outils permet d'écrire et de lancer vos tests unitaires et fonctionnels
à la différence des tests unitaires qui ne cherchent qu'à tester individuellement chacunes des méthodes d'une classe,
les tests fonctionnels vont s'occper de tester l'ensemble d'un processus complexe comme le paiement par carte bleu sur
votre site de e-commerce ou encore l'enegistrement de vos utlisateurs à votre newsletter, etc...
il cherche à tester que tous se passe bien entre un point A et B.
Symfony, Prestashop, Zend, etc.. toutes les rockstars du monde OpenSource en PHP s'en servent.
Site de : PhpUnit SebastianBergmann

Mockery

Utile :

PhpUnit
Mockery est un framework d'objet PHP simulé simple mais flexible à utiliser dans les tests unitaires avec PHPUnit, PHPSpec ou tout autre framework de test. Son objectif principal est de proposer un double framework de test avec une API succincte capable de définir clairement toutes les opérations et interactions possibles sur les objets à l'aide d'un langage DSL (Domain Specific Language) lisible par l'homme.
